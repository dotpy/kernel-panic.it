<base>

Specify a base URL for all relative links in the document. Each link can specify a target frame other than the frame specified by the target attribute.

Attributes

Attributes Type Default Notes
lang %LanguageCode; Backwards compatible. Removed in XHTML 1.1.
xml:lang %LanguageCode;
dir (ltr|rtl) ltr Base directionality (left-to-right or right-to-left) of text.
href %URI; Base URL for relative links in the document.
id ID Fragment identifier (unique within the scope of the document).
target %FrameTarget; Default frame in which to open links. It has been removed in XHTML 1.1.

Example

<head>
  <base href="http://www.kernel-panic.it/images/" />
</head>
<body>
  <!-- The full path of the image is http://www.kernel-panic.it/images/img.gif -->
  <img src="img.gif" />
</body>
