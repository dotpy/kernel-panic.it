<input>

Defines an element that allows the user to enter data.

Attributes

Attributes Type Default Notes
lang %LanguageCode; Backwards compatible. Removed in XHTML 1.1.
xml:lang %LanguageCode;
dir (ltr|rtl) ltr Base directionality (left-to-right or right-to-left) of text.
id ID Fragment identifier (unique within the scope of the document).
class CDATA
style %StyleSheet; An inline style definition.
title %Text; A text to display in a tool tip.
onclick %Script; Script to run on mouse click.
ondblclick %Script; Script to run on mouse double-click.
onmousedown %Script; Script to run on pressing mouse button.
onmouseup %Script; Script to run on releasing mouse button.
onmousemove %Script; Script to run when the mouse pointer moves over the element.
onmouseout %Script; Script to run when the mouse pointer moves out of the element.
onkeypress %Script; Script to run on a key pressed and released.
onkeydown %Script; Script to run on a key pressed.
onkeyup %Script; Script to run on a key released.
accesskey %Character; Sets a keyboard shortcut to access an element.
tabindex %Number; Sets the tab order of an element
onfocus %Script; Script to run when the element gets focus.
onblur %Script; Script to run when the element loses focus.
type (text|password|checkbox|radio|file|
submit|reset|hidden|image|button)		 text Type of input element.
name CDATA Unique name for the element. It is mandatory when the type is button, checkbox, file, hidden, image, password, text or radio.
value CDATA Defines:
  • the text to display on buttons;
  • the (mandatory) value to send for checkboxes and radio buttons on form submit;
  • the default value for hidden, password and text elements;
checked checked Specifies the radio button or checkbox selected by default.
disabled disabled Disables the input field (doesn't apply to hidden fields).
readonly readonly The element value can't be modified by the user (applies only to text fields).
size CDATA Element size (doesn't apply to hidden fields).
maxlength %Number; Maximum number of characters allowed in a text field.
src %URI; Image URL (applies only to type image).
alt CDATA Short description of the image for non-visual browsers.
usemap %URI; Defines the image as a client-side image map.
onselect %Script; Script to execute on element selection.
onchange %Script; Script to run on element value change.
accept %ContentTypes; Comma-separated list of MIME types accepted for file transfer (applies only to type file).

Example

Code Output 
<form action="page.php" method="post">
  <fieldset>
    <legend>Login</legend>
    Username: <input type="text" size="8" /;>
    Password: <input type="password" size="8" /;>
    <br />
    <input type="submit" value="Submit" />
    <input type="reset" value="Reset" />
  </fieldset>
</form>
