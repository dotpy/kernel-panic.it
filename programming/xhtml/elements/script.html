<script>

Defines a script, e.g. Javascript. The language attrbiute, deprecated in HTML 4, has been removed in XHTML 1.0.

Attributes

Attributes Type Default Notes
id ID Fragment identifier (unique within the scope of the document).
charset %Charset Character encoding of the script.
type %ContentType; MIME type of the script.
src %URI; URL of the script.
defer defer Specifies that the script doesn't generate any document content.
xml:space preserve preserve
