<meta>

Allows you to provide meta information about the document, especially useful to search engines. The most common values for the name attribute are:

description
Provides a short description of the document content, displayed by search engines.
author
Specifies the author of the document.
keywords
Specifies some keywords that might help search engines in indexing the document. Unfortunately, many webmasters have long abused this attribute, pushing almost all search engines to ignore it. lang and xml:lang attributes allow you to specify different keywords for each language.
robots
Allows you to prevent search engines from indexing the document. However, it is better to use the robots.txt file, thus avoiding to add a specific tag on each page.
generator
Specifies the application that generated the document.
revised
Specifies the date of the last revision.

The most common values for the http-equiv attribute are:

content-type
Specifies the content type and character set used by the document.
expires
Specifies the document expiration date and time, in order to determine when to fetch an updated copy of the document.
refresh
Forces a page refresh or the load of another page after a certain number of seconds.
set-cookie
Sets the cookie.

Attributes

Attributes Type Default Notes
lang %LanguageCode; Backwards compatible. Removed in XHTML 1.1.
xml:lang %LanguageCode;
dir (ltr|rtl) ltr Base directionality (left-to-right or right-to-left) of text.
id ID Fragment identifier (unique within the scope of the document).
http-equiv CDATA Inserts the value of content inside an HTTP header.
name CDATA Specifies the keyword to which to connect the meta-information.
content CDATA Contains meta-information related to name or http-equiv.
scheme CDATA Provides a more specific format for interpreting the meta-information.

Esempi

<head>
  <meta name="description" content="XHTML 1.0 Reference guide" />
  <meta name="author" content="Daniele Mazzocchio" />
  <meta name="keywords" lang="it" xml:lang="it" content="XHTML,guide,W3C,sviluppo" />
  <meta name="keywords" lang="en" xml:lang="en" content="XHTML,tutorial,W3C,programming" />
  <meta name="robots" content="noindex" />

  <meta http-equiv="content-type" content="text/html; charset=utf-8" />
  <meta http-equiv="expires" content="Tue, 20 Aug 2006 14:25:27 GMT" />
  <meta http-equiv="refresh" content="5; url=http://other.page.it" />
</head>
