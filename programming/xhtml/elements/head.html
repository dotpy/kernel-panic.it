<head>

Provides information about the document which the user agent won't display.

The title element is mandatory; all the others are optional, but there can be only one base element.

Attributes

Attributes Type Default Notes
lang %LanguageCode; Backwards compatible. Removed in XHTML 1.1.
xml:lang %LanguageCode;
dir (ltr|rtl) ltr Base directionality (left-to-right or right-to-left) of text.
id ID Fragment identifier (unique within the scope of the document).
profile %URI; Whitespace separated list of URLs which provide meta-information about the document.
