<object>

Defines an embedded object, to add multimedia elements to the document.

Attributes

Attributes Type Default Notes
lang %LanguageCode; Backwards compatible. Removed in XHTML 1.1.
xml:lang %LanguageCode;
dir (ltr|rtl) ltr Base directionality (left-to-right or right-to-left) of text.
id ID Fragment identifier (unique within the scope of the document).
class CDATA
style %StyleSheet; An inline style definition.
title %Text; A text to display in a tool tip.
onclick %Script; Script to run on mouse click.
ondblclick %Script; Script to run on mouse double-click.
onmousedown %Script; Script to run on pressing mouse button.
onmouseup %Script; Script to run on releasing mouse button.
onmousemove %Script; Script to run when the mouse pointer moves over the element.
onmouseout %Script; Script to run when the mouse pointer moves out of the element.
onkeypress %Script; Script to run on a key pressed and released.
onkeydown %Script; Script to run on a key pressed.
onkeyup %Script; Script to run on a key released.
declare declare Specifies that the object should only be declared, not created or instantiated until needed.
classid %URI; Defines a class ID value.
codebase %URI; URL of the object's code.
data %URI; URL of the object's data.
type %ContentType; MIME types of data specified by the data attribute.
codetype %ContentType; MIME type of the code specified by the classid attribute.
archive %UriList; Whitespace-separated list of URLs to archives. Archives contain resources relevant to the object.
standby %Text; Text to display while loading the object.
height %Length; Height of the object.
width %Length; Width of the object.
usemap %URI; URL of a client-side image map to be used with the object.
name NMTOKEN Defines a unique name for the object (useful within scripts).
tabindex %Number; Sets the tab order of an element
