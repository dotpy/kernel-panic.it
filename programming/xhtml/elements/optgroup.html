<optgroup>

Allows you to logically group the options of a drop-down list created with a <select> element.

Attributes

Attributes Type Default Notes
lang %LanguageCode; Backwards compatible. Removed in XHTML 1.1.
xml:lang %LanguageCode;
dir (ltr|rtl) ltr Base directionality (left-to-right or right-to-left) of text.
id ID Fragment identifier (unique within the scope of the document).
class CDATA
style %StyleSheet; An inline style definition.
title %Text; A text to display in a tool tip.
onclick %Script; Script to run on mouse click.
ondblclick %Script; Script to run on mouse double-click.
onmousedown %Script; Script to run on pressing mouse button.
onmouseup %Script; Script to run on releasing mouse button.
onmousemove %Script; Script to run when the mouse pointer moves over the element.
onmouseout %Script; Script to run when the mouse pointer moves out of the element.
onkeypress %Script; Script to run on a key pressed and released.
onkeydown %Script; Script to run on a key pressed.
onkeyup %Script; Script to run on a key released.
disabled disabled Disables the option group.
label %Text; Text to display to identify the group.

Example

Code Output 
<p>Select a language:
  <select>
    <optgroup label="Functional">
      <option value="lisp">Lisp</option>
      <option value="scheme">Scheme</option>
      <option value="ml">ML</option>
    </optgroup>
    <optgroup label="Logic">
      <option value="prolog">Prolog</option>
    </optgroup>
    <optgroup label="Imperative">
      <option value="fortran">Fortran</option>
      <option value="cobol">Cobol</option>
      <option value="pascal">Pascal</option>
      <option value="c">C</option>
    </optgroup>
    <optgroup label="Object oriented">
      <option value="">C++</option>
      <option value="">Java</option>
    </optgroup>
  </select>
</p>
