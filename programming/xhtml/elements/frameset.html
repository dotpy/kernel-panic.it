<frameset>

Allows you to display multiple documents at the same time, each in its own sub window (frame).

Attributes Type Default Notes
id ID Fragment identifier (unique within the scope of the document).
class CDATA
style %StyleSheet; An inline style definition.
title %Text; A text to display in a tool tip.
rows %MultiLengths; Defines the number and size of rows in the frameset.
cols %MultiLengths; Defines the number and size of columns in the frameset.
onload %Script; Script to execute on page load.
onunload %Script; Script to execute on page unload.

Example

Code 
<frameset cols="25%,*">
  <frame src="frame1.html" />
  <frame src="frame2.html" />

  <noframes>
    <body>
      <p>Your browser doesn't support frames!</p>
    </body>
  </noframes>
</frameset>
