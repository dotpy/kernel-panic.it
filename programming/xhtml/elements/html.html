<html>

This is the root element of the document and, as we've seen before, it must contain an xmlns declaration for the XHTML namespace.

Its content is divided into two parts: one containing information about the document (head, not displayed to the user), and one containing the body of the document (body or frameset), i.e. all the elements displayed by the user agent.

Attributes

Attributes Type Default Notes
lang %LanguageCode; Backwards compatible. Removed in XHTML 1.1.
xml:lang %LanguageCode;
dir (ltr|rtl) ltr Base directionality (left-to-right or right-to-left) of text.
id ID Fragment identifier (unique within the scope of the document).
xmlns %URI; The only legal value is "http://www.w3.org/1999/xhtml".
