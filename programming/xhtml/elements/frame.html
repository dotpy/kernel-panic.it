<frame>

Defines the content of a frame created by the frameset element.

Attributes Type Default Notes
id ID Fragment identifier (unique within the scope of the document).
class CDATA
style %StyleSheet; An inline style definition.
title %Text; A text to display in a tool tip.
longdesc %URI; Detailed description of the frame content. Useful to browsers that don't support frames.
name NMTOKEN Unique name for the frame (useful with Javascript).
src %URI; URL of the document to display in the frame.
frameborder (0|1) 1 Specifies whether drawing a border around the frame or not.
marginwidth %Pixels; Size of the right and left margins of the frame.
marginheight %Pixels; Size of the upper and lower margins of the frame.
noresize noresize Prevents user from resizing the frame.
scrolling (yes|no|auto) auto Defines whether displaying scrollbars or not.

Example

Code 
<frameset cols="25%,*">
  <frame src="frame1.html" />
  <frame src="frame2.html" />

  <noframes>
    <body>
      <p>Your browser doesn't support frames!</p>
    </body>
  </noframes>
</frameset>
