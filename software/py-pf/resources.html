Below is a brief list of references, documentation and books you may find useful depending on your experience with Python, OpenBSD and Packet Filter.

Python links

  • Python Programming Language, the official website
  • Python Tutorial, the best introduction to Python, by Guido Van Rossum himself
  • Python Reference Manual, a detailed description of the syntax and semantics of the Python language
  • Python Library Reference, a complete overview of Python's standard library
  • Dive Into Python, Python from novice to pro, Mark Pilgrim

PF links

  • PF: The OpenBSD Packet Filter, the official Packet Filter FAQ
  • OpenBSD Packet Filter, Daniel Hartmeier's PF page
  • The OpenBSD PF Packet Filter Book - Jeremy C. Reed, Reed Media Services, 2006
  • The Book of PF, 3rd Edition, Peter N. M. Hansteen, No Starch Press, 2014
  • the pf(4) man page
